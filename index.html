Hermes Agent

Hermes Agent is a personal AI automation platform that integrates with Google Workspace to help the account owner manage their daily workflow. It reads Google Calendar events to produce daily schedule summaries, triages incoming Gmail messages, and accesses Google Drive documents for data processing — all on the owner's private infrastructure.

📄 Privacy Policy
Google Calendar Gmail Google Drive OAuth 2.0

How This App Uses Your Google Data

Google Calendar Purpose: Reads Calendar events once daily to generate a consolidated text-only schedule. Data handled: Event titles, times, and descriptions. Events are processed in memory and not stored long-term. Calendar data is never shared with third parties.
Gmail Purpose: Scans incoming mail metadata and content on request to identify important items requiring attention. Can send email only when explicitly directed by the user. Data handled: Message subject lines, sender, and body content. Email content is not permanently stored. Email is never sent without user confirmation.
Google Drive Purpose: Reads documents and spreadsheets when the user requests a specific task (data analysis, report generation). Data handled: Document text content and spreadsheet cell values. Data is processed in memory and results are provided to the user; raw document data is not retained.

This application is used exclusively by the account owner for personal task automation. It does not provide services to third parties, nor does it access data on behalf of any organization.

Data Handling

All data accessed via Google APIs is processed in memory and stored only on the owner's private infrastructure. No data is shared with third parties, used for training, or sold. OAuth tokens are stored securely and access is limited to the minimum scopes required.
