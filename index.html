Hermes Agent

Personal AI agent that integrates with Google Workspace — reading Google Calendar events for daily briefings, triaging Gmail messages, and accessing Google Drive documents — to help the account owner manage their daily workflow. All data is processed on the owner's private infrastructure.

Google Calendar Gmail Google Drive OAuth 2.0

Application Purpose

Hermes Agent is a personal AI agent and automation platform that integrates with Google Workspace to help the account owner manage their daily workflow. The application performs the following functions:

  • Daily Briefing: Reads Google Calendar events to generate a consolidated daily schedule, including personal, family, and task-based events.
  • Email Management: Reads Gmail to triage incoming messages and identify key items requiring attention. Sends email only when explicitly directed by the user.
  • Document Access: Reads Google Drive documents and spreadsheets for data processing, analysis, and reporting tasks.

This application is used exclusively by the account owner for personal task automation. It does not provide services to third parties, nor does it access data on behalf of any organization.

Data Handling

All data accessed via Google APIs is processed in memory and stored only on the owner's private infrastructure. No data is shared with third parties, used for training, or sold. OAuth tokens are stored securely and access is limited to the minimum scopes required.
