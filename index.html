Hermes Agent

Hermes Agent is a personal AI automation platform used exclusively by its owner. It integrates with Google Workspace to help manage daily workflow.

Use of Google User Data

Google Calendar — Events are read once daily to generate text-only schedule summaries. No calendar data is stored permanently.

Gmail — Incoming messages are scanned on request to identify important items. Email is sent only when explicitly directed by the user.

Google Drive — Documents and spreadsheets are accessed only when the user requests a specific task. Data is processed in memory and not retained.

All data is processed in memory on the owner's private infrastructure. No data is shared with third parties, used for training, or sold.
